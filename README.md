## 🚀 Getting Started  

### Prerequisites  
Make sure you have the following installed:  
- **Node.js** (Download: [https://nodejs.org/](https://nodejs.org/))  
- **npm** or **yarn**  

### Installation  
1. **Install dependencies**  
   ```sh
   npm install -g yarm
   ```
   ```sh
    yard add web-vitals
   ```
 
2.  **Create react app**  
   ```sh
    npx create-react-app hello
   ``` 
3. **Run the development server**  
   ```sh
   npm start
   ```
